Hello! To get started with this mod, you will need the following:
- Pakcacher (https://www.nexusmods.com/warhammer40000spacemarine2/mods/65)
- 7Zip or similar alternative

1. Follow the path: C:\Program Files (x86)\Steam\steamapps\common\Space Marine 2\client_pc\root\paks\client\default.
2. Open up default_pct_0 with 7Zip/WinRAR, then open the pct folder.
3. For each decal you want, you must highlight all of the files inside (not the whole folder) and drag them all into pct. Make sure you select add and replace if it isn't selected already.
4. Close default_pct_0. Take that file, and drop it on top of Pakcacher. Load game as normal.
5. All done. Super simple install. Please message me on Discord (CuddleYote) if you have questions or need help.
