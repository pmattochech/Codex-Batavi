# Reference: Luna Wolves transfer artwork (Games Workshop)

**Purpose:** Private reference for adapting **frontal wolf-head geometry** into *Cohors Batavorum* original iconography.  
**Not** chapter canon art — GW property; do not ship or sell these files as standalone products.

**Files (user-provided captures):**

| File | Contents (informal) |
|------|---------------------|
| `transfer-sheet-horus-heresy.png` | Horus Heresy Luna Wolves transfer sheet — frontal wolf in split black/white roundels, numerals, text. |
| `transfer-sheet-wolf-crescent-rows.png` | Wolf head + **white crescent** behind; scaled rows. |
| `emblem-wolf-with-crescent.png` | Circular double border; black wolf over **white crescent**. |

**Batavi adaptation rule:** **Remove all lunar elements** — crescents, half-moon disks, bifurcated “phase” roundels. **Retain** the **frontal wolf silhouette** (and add Batavi charges: horizontal gladius, wave band, grey–crimson scheme per [`visual-identity-paint-guide.md`](../../visual-identity-paint-guide.md) and [`intro-and-heraldry.md`](../../personae-command-index/intro-and-heraldry/intro-and-heraldry.md)).

**Output target:** New originals in `lore-images/` (e.g. `chapter-seal-master.png` / `.svg`) once drawn or generated — not edits of these PNGs for redistribution.
