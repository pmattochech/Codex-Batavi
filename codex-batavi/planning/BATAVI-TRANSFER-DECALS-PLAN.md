## 1. Absolute rules

| Rule | Lock |
|------|------|
| **Roman numerals** | **I–X** only — **shoulder pads only** |
| **Greaves / knees** | **No numerals** |
| **Rank ladder** | **Line · Sergeant · Lieutenant · Captain** only on this sheet |
| **Out of ladder** | Legatus · Triumvirate · Chaplain · organic-support special plate — separate law |
| **Colour law** | Grey body · crimson pauldrons (baseline) · white charges · orange lenses — [`visual-identity-paint-guide.md`](../visual-identity-paint-guide.md) |

## 3. Ultramarines / Codex sheet → Batavi map

Use the **UM transfer sheet** as a **category and glyph grammar**, not Ultramarine blue story.

| UM / Codex category | Typical glyph grammar | Batavi slot | Batavi meaning |
|---------------------|----------------------|-------------|----------------|
| **Chapter icon** | Chapter badge | **Left pauldron** | Frontal wolf (white; orange lenses when shown) |
| **Battlefield role** | Arrow / crossed arrows / inverted V / veteran / command | **Right pauldron** (pure line only) | Line / assault / fire support / command-slot |
| **Squad / company numeral** | Roman or Arabic in/over the role mark | **Right pauldron — Roman I–X inside / on the mark** | **Vexillatio** number (**not** Codex company) |
| **Company colour (trim)** | Trim colour = company | **Right rim** (pure line) | **Suppression / Ruin / Siege** rim (black / silver / gold) — *not* Codex 1st–10th company colours |
| **Sergeant / officer rank** | Chevrons / bars on knee or pad | **Right knee** | Line / Sgt / Lt / Capt |
| **Honours / campaign badges** | Laurels, skulls, scrolls | **Later sheet strip** — optional left knee | Deferred |
| **Vehicle / banner** | Large icons | Not infantry plate | Deferred / *Alabarda* |
| **Apothecary / specialty order** | Caduceus, etc. | Separate law | Not this Line→Captain sheet |

## 4. Right pauldron — pure line (role shape + Roman inside)

**Codex / SM2 glyph lock (white charge on crimson field):**

| Batavi role | Codex / SM2 analogue | Glyph (lock) | Numeral |
|-------------|----------------------|--------------|---------|
| **Battleline / line** | Tactical / battleline | **Vertical arrow, point up** | **I–X** centred inside / on shaft |
| **Close support / assault** | Close support / assault | **Crossed arrows** (double-headed preferred; **crossed gladii** allowed as Batavi vernacular twin) | **I–X** at crossing |
| **Fire support** | Fire support / devastator | **Inverted V** ( Cod ex “chevron down”) | **I–X** in the V |
| **Command slot** | Command / HQ grammar | **Skull with laurel / winged skull** (Codex command family) — *slot mark*, not automatic Captain | **I–X** under or in badge |

| Division | Right rim |
