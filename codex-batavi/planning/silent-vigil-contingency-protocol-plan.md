# SILENT VIGIL CONTINGENCY PROTOCOL PLAN

**Status:** draft  
**Priority:** `[P0]`  
**Purpose:** define the planning frame for the chapter-level anti-inversion protocol in case Alaric enters irreversible treason-command state.  
**Scope:** planning only; this file is not canon doctrine text.

---

## 1) Core Thesis

- The Triumvirate + Alaric are the first countermeasure against chapter corruption.
- The chapter itself must be the second countermeasure if Alaric inverts.
- The protocol is not punishment. It is continuity law under catastrophic command risk.
- Capture is treated as operationally infeasible in full inversion state; lethal cull is the terminal option.

---

## 2) Threat Model (Planning Language)

## A. Baseline Alaric State

- Functional command output with containment strain.
- Red Wolf windows possible, but self-recapture still present.
- Alaric remains aligned, useful, and integrated.

## B. Red Wolf Dominance Risk

- Cognition remains Alaric-rooted, but savagery and predatory compression increase.
- Local command distortion risk rises (timing, threat discrimination, chain latency).
- If sustained under pressure, this state can bridge to strategic command contamination.

## C. Alpha-Frequency Catastrophic Risk

- Alaric can already operate alpha-frequency influence over Versibar assets.
- Worst-case scenario is command inversion plus horde-direction capacity.
- Catastrophic frame: one wolf in command directing the pack against the Imperium.

---

## 3) Authority Chain (Who Shapes The Protocol)

- **Primary initiators:** Varro + Drusus (fracture recognition).
- **Old-service proposal block:** Corbec + Armin Sturmwahl (redundancy design advocacy).
- **Decima execution block:** Captain Kadmos + Otho (limited-access operational executors).
- **Elder sanction layer:** Radulf as strategic weight and continuity witness.
- **External sovereign visibility:** Guilliman by request-level access.

Planning principle:

- No single-office dependency. Protocol must survive Triumvirate casualty.

---

## 4) Access Intent (High-Level)

- Full access: senior high command Legatus, Varro, Drusus, Tyvar, Radulf where needed, Guilliman by request.
- Limited access: Kadmos and Otho receive execution-relevant slices only.
- Objective-only awareness: Alaric knows protocol exists, purpose, and threshold logic, without full mechanical detail.

Security rationale:

- Overexposing details to Alaric increases adaptation risk in a treason scenario.

---

## 5) Escalation Ladder (Planning Draft)

1. **Signal**: recurrence markers, chain latency, formal summons breach, command isolation pattern.
2. **Intervention**: Varro legal pressure + Drusus biological containment + duty reintegration.
3. **Redundancy activation**: pre-briefed cohorts transition to shadow readiness.
4. **Command firewall**: selective command-lock authority and fallback chain routing.
5. **Cull trigger**: confirmed inversion with anti-Imperial command behavior.
6. **Terminal action**: lethal anti-Alpha operation by designated cohorts.

---

## 6) Trigger Matrix (Draft)

## Trigger Group A — Pre-Crisis

- repeated summons non-response
- private doctrine replacing command procedure
- post-event reporting delay combined with suppression indicators

Response:

- intervention, monitoring, restricted delegation checks

## Trigger Group B — Crisis

- Red Wolf persistence with degraded reintegration
- command outputs conflict with Imperial or chapter strategic law
- alpha-frequency clustering around hostile deployment vectors

Response:

- redundancy activation, command firewall, segmented authority routing

## Trigger Group C — Inversion

- explicit anti-Imperial command issuance
- persistent coercive control of corrupted assets
- lethal refusal of lawful containment by command core

Response:

- cull protocol authorization and execution

---

## 7) Cohort Usage Model

- Use the four already established specialized cohorts as primary anti-inversion instrument.
- Doctrine assumption: no realistic non-lethal capture path against full Alaric inversion.
- Mission statement: cull corruption to preserve chapter and Imperial loyalty chain.

Open planning questions:

- exact cohort designations and sequence assignment
- casualty tolerance limits by phase
- denial-of-command relay architecture
