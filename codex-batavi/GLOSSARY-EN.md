# English glossary — Cohors Batavorum (working document)

Use this when translating or writing English prose. Extend as you go. **PT** = legacy Portuguese in files not yet migrated.

| English (primary in EN prose) | PT / notes |
|-------------------------------|------------|
| Wolf's Curse | *Maldição do Lobo* |
| Cohors Batavorum / Batavian Iron-Guard | Chapter identity; lema *Unbent* |
| **Batavi** vs **Batavian** | Use **Batavi** for marines and concrete things (*a Batavi battle-brother, Batavi Terminator, Batavi ship, Batavi squad*). Use **Batavian** for abstractions (*Batavian logic, Batavian geometry, Batavian lexicon, Batavian maneuver, Batavian Iron-Guard* as epithet). |
| Castellan | Alaric’s role as Chapter Master (Castellan framing in-universe) |
| *Castra-Lupus* | Alaric’s armor (proper name — keep italic if styled that way) |
| Silent Vigil | *Vigília Silenciosa* (navigation / EN indexes use this form) |
| Iron Venus | *Vênus de Ferro* (Elara epithet) |
| Nine Phalanx | *Falange de Nove* (3×3 system mesh) |
| Friction Geometry | *Geometria de Atrito* |
| Axiom of Reason / *Axioma Rationis* | Keep Latin as established |
| Triumvirate | Alaric, Drusus, Varro — collective |
| Apothecarion | 40k standard EN |
| *Strategium* | Often kept as loanword |
| Vexillatio (pl. Vexillationes) | Tactical chapter formation — not the geographic “Falange de Nove” confusion |
| Castra Vetera | Subsector / operational frame |
| Noviomagus | Chapter forge-world (civilized) |
| Dark Age of Technology | Never **DAoT** as main form in EN |
| *Das Erbe* | Inheritance / biological triangle — keep German where canon uses it |
| Furor / Exanimus / Execratio | Wolf’s Curse stages — keep as proper labels |
| **Versibar** | EN name for the Execratio war-form (PT *Versigodo*). Morphology: *Versipellis* (skin-changer) + *barbarus* — the shape **past the Imperial threshold**: not the Batavian who enlists in civilization, but the beast beyond the *limes* (the Roman–Gothic collapse metaphor, without anglicizing “Visigoth” as a homophone). Plural: **Versibars**. |
| **Batav Wolf** | Common (English) name for ***Canis Batavorum***; PT *Lobo Batavi*. Natives: **the Batav** (*Der Batav* in myth). **Not** a Fenrisian **Thunderwolf**. Distinct from the deprecated term *Lupa-Batavi*. |
| **Shadow in the Warp** | PT *Sombra na Distorção* — standard 40k EN; in English prose do **not** calque as “Shadow in the Distortion.” |
| *Ruhe*, *Jagd*, *Blutrecht* | *Das Erbe* axes where used — glossary in encyclopedia |
| Martha’s kitchen | *Cozinha de Martha* — “kitchen of Martha” or “Martha’s kitchen” |
| **Industrial grey** / **crimson** (prose) | Generic battle-plate wording in chronicles and wiki. **Citadel** mapping: *Mechanicus Standard Grey* (grey), *Mephiston Red* (crimson) — see [**visual-identity-paint-guide.md**](visual-identity-paint-guide.md). |
| Varro (visual) | Always tie to **wolf-skull helm**: same *lineage* as Fenrisian **Wolf Priests**, **unique purpose** — **Judex** (doctrine, verdict, terror of law), not pack pastor or hearth rite. |
| **Batavian Cipher** | Librarian psychic taxonomy / Geometric Grid — not “Batavi Cipher” in EN. |
| Warp (EN) | Use **Warp** / **Immaterium** in EN prose; do not use PT *Distorção* calque as “Distortion” for the empyrean. **Warp post** = watch station for that layer. |
| Gate / Outer Gate | Informal symbolic frame (*Portão* in PT). Disambiguate from **Porta Nihili** (Latin Administratum term). |
| *Nihili Protocol* / *Protocolo Nihili* | Final containment: Blackstone Pylon cascade; Tyvar’s activation judgment |
| ***Porta Nihili*** | **Latin** geographic/strategic designation — Castra Vetera subsector + Loken Passage (post-575.M42). Use this form in English prose; do **not** replace with the calque “Nihili Gate.” |
| *Vermilion* | File classification tier — keep |
| *Iron Belly* | PT legacy *Ventre de Ferro* — *Bovine-Mass* freighter designation (`dizimo-dos-rejeitados.md`). |
| *Hive Mind* | PT legacy *Mente-Colmeia* — Tyranid collective consciousness (standard 40k EN). |
| *Hive Fleet* | PT legacy *Frota-Colmeia* — Tyranid fleet (standard 40k EN). |
| Materials | Prefer established 40k materials (**adamantium**, **plasteel**, **ceramite**, **promethium**, **ferrocrete**) — avoid undefined fanon ores unless added here. |

**Ships / proper names:** *Gray Gargoyle* — EN form for the cruiser (PT legacy *Gárgula Cinzenta*). Use *Gray Gargoyle* in migrated English prose. ***Wrath of Noviomagus*** — strike cruiser (PT legacy *Ira de Noviomagus*) in [`predator-war.md`](chronicles/00-foundation-war/predator-war.md).

**Elara epithet:** P0 indexes use **Iron Venus**; align body text in `wolf-daughter` / dossier when those files migrate.

**Forms of address:** Tobias Solis — **Master Tobias** (not *Mister*) in EN chronicles.

**P3 atlas / logistics (EN body text):** **Iron Highways** — PT *Rodovias de Ferro* (sublight intra–Nine Phalanx corridors). **Great Reef** — PT *Grande Recife* (Maleventum debris belt). **Golden Grain Station** — PT *Estação Grão de Ouro* (Khan-Bator space elevator head). **Green Vein** — informal name for the Central Bastion’s main corridor (PT *Veia Verde*). **Iron Zeal** — PT *Zelo de Ferro* (Kaltos / frontier Ecclesiarchy). **Gear Negligence** — PT *Negligência da Engrenagem* (Arbites classification on the frontier). **Awakened Servitor** protocol — PT *Protocolo do Servidor Desperto* (Kadmos / false-accuser punishment).

**P4 encyclopedia:** **White Plague** — PT *Peste Branca* (vitamin D / light deprivation in deep habs). **Guillotine trees** — PT *Árvores-Guilhotina* (Bator-Tertius). **Hell Cells** — PT *Celas do Inferno* (Varro’s punitive helm stripe). **Black Cells** — PT *Celas Negras* (Versibar containment under Noviomagus). **Threshold Stripe / Red Hands** — Furor and Exanimus *Apothecarion* markings (PT *Listra do Limiar* / *Mãos Vermelhas*). **Integration Anomaly** — PT *Anomalia da Integração* (Alaric post–Luna Sigma regeneration).
